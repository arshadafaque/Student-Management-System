# Student-Management-System



Introduction

This is a simple Spring Boot application that demonstrates CRUD (Create, Read, Update, Delete) operations using JpaRepository.

Technologies Used

-Java 17+

-Spring Boot

-Lombok

-Spring Data JPA

-Hibernate

-Oracle DataBase

-Maven

Features

-Create a new entity

-Retrieve entity by ID

-Retrieve all entities

-Update an entity

-Delete an entity
